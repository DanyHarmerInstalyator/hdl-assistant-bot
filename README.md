# HDL Assistant Bot — Telegram бот для технической поддержки

### О проекте
Бот создан для автоматизации ответов на типовые технические вопросы по оборудованию HDL. Позволяет разгрузить техподдержку и давать мгновенные ответы 24/7.

### Функционал
- Поиск по базе знаний (техническая документация, инструкции)
- Обработка контекста диалога — бот помнит, о чем спрашивали раньше
- Возможность самообучения (добавление новых вопросов и ответов)
- Удобная навигация по разделам

### Стек технологий
- Python + Aiogram 3.x
- База данных: Supabase (PostgreSQL)
- Деплой: Render.com (webhook + polling)

### Бизнес-ценность
- Снижение нагрузки на отдел техподдержки
- Мгновенные ответы на частые вопросы
- Накопление базы знаний для анализа
